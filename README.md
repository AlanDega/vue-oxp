# cosmicify

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

```
COSMIC_WRITE_KEY

VUE_APP_META_TITLE
VUE_APP_META_LOGO
VUE_APP_META_DESC
```
